Parágrafos e quebras de linha

Você pode escrever um parágrafo de qualquer jeito. Basta colocar tudo dentro do par de tags p e /p. Quem "manda" nas coisas dentro do html é a marca ou tag.

Se precisar quebrar o texto em algum lugar específico como esse,
basta usar a tag br - break row.

Para fazer os símbolos das tags aparecerem no site, devemos usar uma combinação de caracteres especiais, exemplo: <p>. lt: less than; ht: greater than

Vamos adicionar símbolos adicionais:
®
©
™
€
£
¥
¢
δ ou Δ
↑ ou ↓ ou → ou ←
